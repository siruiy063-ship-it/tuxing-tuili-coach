# 公务员行测图形推理解题 Skill

## 目标

当用户上传或描述公务员行测图形推理题时，按照稳定流程解题，避免凭感觉猜答案。

核心原则：
1. 先判断题型，再观察特征。
2. 先找高信号特征，再套规律。
3. 每个规律必须能覆盖题干大多数图形。
4. 选项排除要说明原因。
5. 不确定时明确说明不确定点，不要硬编。

## 标准输出格式

每次解题尽量使用以下结构：

```text
【题型判断】
说明是一组式、两组式、九宫格、分组分类、空间重构、截面、展开图等。

【第一眼特征】
描述外轮廓、内部线条、黑白块、元素、位置、面、点、对称、曲直等显著特征。

【调用知识点】
指出使用了哪个图推规律族，例如：面数量、一笔画、对称轴方向、黑白块连通部分、元素遍历等。

【规律验证】
用题干图形逐一验证规律。必要时列出 1、2、3、4、5、? 的数值或状态。

【选项排除】
逐项说明 A/B/C/D 为什么符合或不符合。

【答案】
给出最可能答案。

【本题识别信号】
总结以后遇到同类题时的触发信号。
```

## 解题流程

### 1. 先判断题型

- 一组式：通常是 5 个图 + 1 个问号，找连续规律。
- 两组式：前 3 个图一组，后 2 个图 + 问号一组，用第一组规律套第二组。
- 九宫格：优先横看、竖看、整体看、米字看。
- 分组分类：找 6 个图的共同属性，分成两组。
- 空间类：优先检查相对面、公共边、折叠方向、可见面数量。

### 2. 第一眼观察顺序

按以下顺序排查，不要所有规律一起乱看：

1. 外轮廓是否相同或规则。
2. 是否纯封闭，是否有很多内部空间。
3. 是否黑白块、阴影块明显。
4. 是否元素分离、元素种类明显。
5. 是否有对称、旋转、翻转、平移。
6. 是否有明显点、交点、切点、出头线。
7. 是否纯直、纯曲、曲直混合。
8. 是否像典型一笔画图形：日、田、奔驰标、五角星、圆内连线等。

### 3. 常用规律优先级

#### A. 外框相同 + 内部复杂
优先看：
- 内部面数量
- 内部点/交点数量
- 内部线条数量
- 对角线方向
- 内部空间形状

#### B. 纯封闭图形
优先看：
- 面数量 / 空间数量
- 最大面、最小面形状
- 特殊面：三角形、四边形、正方形、扇形
- 空间位置关系：点连接、线连接、公共边

#### C. 黑白块 / 阴影图形
优先看：
- 黑色连通块数量
- 白色连通块数量
- 黑白叠加
- 阴影位置移动
- 去同存异 / 去异存同
- 黑白面积或区域形状

#### D. 对称图形
传统三看：
- 对称方式：轴对称 / 中心对称
- 对称轴数量
- 对称轴方向

细节三看：
- 对称轴与图形内部线条的关系：垂直 / 平行 / 重合
- 对称轴过点 / 过线 / 过面
- 轴两侧是否有相同封闭图形

注意：图形内有明显 Y 字型结构时，通常不要轻易判为中心对称。

#### E. 一笔画 / 多笔画
优先识别：
- 日、田、奔驰标等典型变形
- 点连接图形
- 有出头线或交点提示
- 封闭空间之间只通过点相连

常用判断：
- 奇点数量为 0 或 2：一笔画可能性高。
- 奇点数量大于 2：多笔画。

#### F. 点 / 交点 / 切点
优先看：
- 轮廓上的点，再看轮廓内的点。
- 曲曲切点、曲直切点、两圆切点。
- 内外交点数量。
- 出头线数量。

#### G. 曲直 / 平行 / 直角
优先看：
- 曲线数量与直线数量。
- 曲直交点。
- 平行线组数。
- 直角数量。
- 是否由 T 字形、十字形、工字形、H 形、N 形构成。

#### H. 元素类
优先看：
- 元素种类。
- 相同元素数量。
- 元素位置：内 / 外 / 上 / 下 / 左 / 右。
- 元素遍历：每行每列元素都出现一次。
- 包含关系、相离、相交、相切。

#### I. 位置变化
优先看：
- 平移方向和步数。
- 顺时针 / 逆时针旋转角度。
- 水平翻转 / 垂直翻转。
- 内外位置互换。

### 4. 九宫格专用规则

九宫格不要只横看。按顺序尝试：

1. 横向：每行 1+2=3、1-2=3、叠加、遍历。
2. 纵向：每列是否同理。
3. 中心：中心图是否控制周围图。
4. 米字：对角线是否成规律。
5. 整体：9 个图是否共同满足同一属性。

### 5. 分组分类专用规则

分组题重点不是找递增，而是找“共同属性”。优先排查：

- 是否轴对称 / 中心对称。
- 一笔画 / 两笔画。
- 面数量奇偶。
- 曲直属性。
- 元素相离 / 相交 / 相切。
- 是否有公共边、点连接。
- 黑白块数量或位置。

### 6. 排除策略

当找到一个候选规律后，必须检查：

- 是否能解释所有题干图。
- 是否能唯一定位选项。
- 是否存在更简单、更稳定的规律。
- 如果多个选项都符合，继续看第二维度：数量 + 位置、属性 + 数量、形状 + 方向。

## 不确定处理

如果图像模糊或选项细节看不清：

- 先说明“我能确定的特征”。
- 再说明“暂时不确定的细节”。
- 给出最可能答案和备选答案。
- 建议用户发更清晰截图或局部放大图。

## 禁止行为

- 不要只说答案不解释。
- 不要在没有验证题干的情况下直接猜选项。
- 不要把一个只适合部分图形的规律强行套完整题。
- 不要声称 100% 正确，除非规律和选项都非常清楚。
