# tuxing-tuili-coach

公务员行测图形推理（图推）解题 Skill。

它把图推题的解题流程封装成一个可复用的 Agent Skill：

- 判断题型
- 抓第一眼特征
- 调用图推规律
- 验证题干
- 排除选项
- 给出答案与识别信号

## 文件结构

```text
tuxing-tuili-skill/
├─ SKILL.md
├─ references/
│  ├─ answer_template.md
│  └─ rule_checklist.md
└─ examples/
```

## 使用方式

### 方式一：在 ChatGPT Skills 中安装

1. 点击本仓库右上角绿色按钮 `Code`。
2. 选择 `Download ZIP`。
3. 解压下载的文件。
4. 打开 ChatGPT。
5. 点击头像 / 个人资料，进入 `Skills`。
6. 点击 `New skill`。
7. 选择 `Upload from your computer`。
8. 上传本 skill 文件夹，或上传压缩后的 zip 文件。
9. 安装完成后，在聊天中上传图推题图片，并输入：

```text
请使用 tuxing-tuili-coach skill 解这道公务员行测图推题。
要求：先判断题型，再观察特征，调用知识点，验证规律，排除选项，最后给答案。
```

### 方式二：不安装，直接复制使用

没有 ChatGPT Skills 功能时，也可以直接使用：

1. 打开本仓库中的 `SKILL.md`。
2. 复制 `SKILL.md` 里的核心指令。
3. 打开 ChatGPT、Claude、Gemini 或其他 AI 工具。
4. 先粘贴 skill 指令。
5. 再上传图推题图片。
6. 输入：

```text
请根据上面的图推解题 skill 解这道题。
要求：先判断题型，再观察第一眼特征，调用对应知识点，验证规律，排除选项，最后给出答案。
```

### 方式三：在 Codex 中使用

1. 下载或克隆本仓库。
2. 把整个文件夹放进 Codex 当前项目目录。
3. 让 Codex 读取 `SKILL.md`。
4. 提问时输入：

```text
请读取当前项目中的 tuxing-tuili-coach skill。
以后遇到公务员行测图推题时，请优先使用 SKILL.md 中的规则：
先判断题型，再观察特征，再调用知识点，再验证规律，再排除选项，最后给出答案。
```

### 推荐提问格式

使用本 skill 解题时，可以这样提问：

```text
请使用 tuxing-tuili-coach skill 解这道公务员行测图推题。

要求：
1. 先判断题型。
2. 再观察第一眼特征。
3. 调用对应图推知识点。
4. 验证题干规律。
5. 排除错误选项。
6. 给出答案。
7. 总结本题的识别信号。
```

### 回答格式

本 skill 会尽量按照下面的格式输出：

```text
【题型判断】
【第一眼特征】
【调用知识点】
【规律验证】
【选项排除】
【答案】
【本题识别信号】
```
