# 图推规律排查清单

## 观察入口

- 外轮廓相同：看内部点、线、面、对角线、空间形状。
- 纯封闭：看面数量、最大/最小面、特殊面形状、空间位置。
- 黑白块明显：看黑/白连通块数量、叠加、位置移动、去同存异。
- 元素分离：看元素种类、数量、位置、遍历。
- 线条突出：看直线数、曲线数、交点、平行、直角、出头线。
- 对称明显：看方式、轴数、轴方向、轴与内部线关系。
- 点连接明显：考虑一笔画、连接方式、公共边。

## 验证原则

1. 先用题干验证，不先看选项。
2. 规律要能解释所有已知图形。
3. 如果多个选项符合，叠加第二规律。
4. 最后总结识别信号，方便用户积累题感。
